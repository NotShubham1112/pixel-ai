# Security Policy

## Supported Versions

Only the latest version of Pixel is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within Pixel, please do not open a public issue. Instead, report it privately to the maintainer.

All security vulnerabilities will be promptly addressed.
